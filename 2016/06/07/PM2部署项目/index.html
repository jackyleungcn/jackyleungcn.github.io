PM2部署项目
2016-06-07 17:06   folder 服务端 label Node.js

PM2 http-server 部署

1.cd到项目根目录
2.which http-server 找到http-server模块路径
3.pm2 start /usr/local/bin/http-server --name PM2_NAME -- -p PORT -d false
（PS: pm2 start /usr/local/bin/http-server --name static-server -- -p 8080 -d false）

PM2 npm 部署

pm2 start npm --name pm2_name -- start

PM2 常用命令

pm2 list //查看PM2项目列表

pm2 start  //启动 全部/某个 项目

pm2 stop  //停止 全部/某个 项目

pm2 restart  //重启 全部/某个 项目

pm2 delete  //删除 全部/某个 项目

pm2 logs //查看全部项目日志

pm2 logs  //查看某个项目日志

pm2 save //保存当前项目列表配置

pm2 resurrect //恢复项目列表，数据来自上一次pm2 save
